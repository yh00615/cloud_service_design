# Draw.io 라이브러리 파일

이 폴더에는 AWS 아키텍처 다이어그램 작성을 위한 draw.io 라이브러리 파일이 포함되어 있습니다.

## 파일 목록

- `aws-architecture-icons.xml` - AWS 공식 아키텍처 아이콘 라이브러리
- `custom-aws-components.xml` - 실습용 커스텀 컴포넌트 (선택사항)

## 사용 방법

1. draw.io (https://app.diagrams.net/) 접속
2. **File** > **Open Library from** > **Device** 선택
3. 다운로드한 `.xml` 파일 선택
4. 왼쪽 패널에 라이브러리가 추가됨

## 다운로드 링크 형식

실습 가이드에서 다음과 같이 링크를 제공합니다:

```markdown
> [!DOWNLOAD]
> [aws-architecture-icons.xml](/files/drawio-libraries/aws-architecture-icons.xml)
>
> - AWS 공식 아키텍처 아이콘 라이브러리
>
> **관련 태스크:**
>
> - 태스크 1: draw.io에서 라이브러리를 불러와 아키텍처 다이어그램을 작성합니다.
```

## 파일 명명 규칙

- 소문자 + 하이픈 사용
- 확장자: `.xml`
- 예시: `aws-architecture-icons.xml`, `custom-components.xml`
